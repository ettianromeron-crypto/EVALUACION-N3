EL SIGUIENTE CODIGO TIENE EL PROPOSITO DE SER UNA PAGINA WEB SIMPLE CON
UN FORMULARIO QUE GUARDE LOS DATOS INGRESADOS Y COMPRUEBE LOS DATOS DENTRO
DE ESTE.

SE LOGRO REALIZAR LA PAGINA WEB CON UN ESTILO "STYLE.CSS" Y UNA LOGICA
INTERNA SIMPLE "SCRIPT.JS".
NO SE LOGRA EL OBJETIVO DE GUARDAR LOS DATOS DENTRO DE UNA BASE DE DATOS 
POR EL DESCONOCIMIENTO DEL TEMA.
